# 机械发明家 - 零件的奇妙配对

## 项目介绍

"机械发明家"是一个专注于机械零件教育的互动网站，旨在通过有趣的方式帮助用户学习和了解各种机械零件的知识。网站采用现代化的设计风格，结合3D模型展示和互动游戏，为用户提供沉浸式的学习体验。

## 功能特点

### 1. 零件档案馆
- 分类展示各种机械零件，包括连接类、传动类、支撑与导向类、弹簧与弹性元件等
- 每个零件都有详细的介绍，包括所属分类、核心作用、工作原理、常见类型、应用场景等
- 支持3D模型查看，可旋转、缩放查看零件细节
- 提供搜索功能，方便快速找到特定零件

### 2. 帮零件找名字
- 互动游戏形式，通过识别零件来学习其名称
- 提高用户对机械零件的认知能力

### 3. 零件小考验
- 测试用户对机械知识的掌握程度
- 提供不同难度的测试模式
- 测试完成后给出详细解析和评分

### 4. 每日一题
- 每天更新一道机械相关的问题
- 包含详细的解析和答案
- 帮助用户持续学习机械知识

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript
- **3D模型**：使用Model Viewer库实现3D模型展示
- **响应式设计**：适配不同屏幕尺寸
- **动画效果**：使用CSS动画和过渡效果增强用户体验

## 如何运行

1. 克隆或下载项目到本地
2. 使用本地服务器打开项目（如Python的`python -m http.server 8000`）
3. 在浏览器中访问 `http://localhost:8000` 即可

## 项目结构

```
├── draco/              # 3D模型压缩库
├── images/             # 网站图片资源
├── model/              # 3D模型文件
├── model1/             # 额外的3D模型文件
├── model2/             # 分类整理的3D模型文件
├── DRACOLoader.js      # 3D模型加载器
├── GLTFLoader.js       # GLTF格式模型加载器
├── three.min.js        # Three.js库
├── index.html          # 首页
├── main-content.html   # 主内容页
├── archives.html       # 零件档案馆
├── match-names.html    # 帮零件找名字游戏
├── parts-test.html     # 零件小考验
├── connection-parts.html   # 连接类零件详情
├── transmission-parts.html # 传动类零件详情
├── support-parts.html      # 支撑与导向类零件详情
├── spring-parts.html       # 弹簧与弹性元件详情
├── other-parts.html        # 其他常用机械零件详情
├── search-results.html     # 搜索结果页
└── README.md               # 项目说明文档
```

## 如何使用

1. **首页**：点击"了解详情"进入主内容页
2. **主内容页**：选择不同的功能模块（零件档案馆、帮零件找名字、零件小考验）
3. **零件档案馆**：浏览不同分类的零件，点击零件查看详细信息和3D模型
4. **帮零件找名字**：通过互动游戏学习零件名称
5. **零件小考验**：完成测试题，查看解析和评分
6. **每日一题**：查看当天的问题，思考后翻转卡片查看答案和解析
